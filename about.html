Core Functional Modules

The website is built around decision speed and explainability.

  • Dashboard: composite score, regime, alerts, top drivers, and key indicators.
  • Daily Report: structured narrative, AI interpretation, and archived daily links.
  • Indicators & Glossary: full indicator dictionary and model terminology.
  • Portfolio Watchlist: user watchlists, position mapping, and macro-linked risk output.
  • Regime & Transmission: regime engine, geopolitical overlay, action bias, and transmission heatmap.
  • Stock Prediction: ticker-level forecasts, backtest curves, and feature importance.

What You Get As A User

  • A single macro regime view that is updated every day.
  • Transparent logic from data -> signal -> action.
  • Portfolio-level risk guidance connected to your own holdings.
  • Bilingual interface (Chinese/English) for teams and clients.

Daily Operating Workflow

01

08:00 (China Time): Data Refresh

02

09:00 (China Time): Report Generation

Run model scoring, produce daily report, and write outputs to database.

03

Post Report: Distribution & AI

Send report emails, run AI interpretation, and sync summary back to dashboard.

System Architecture (High Level)
